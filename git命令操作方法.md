vuecss项目
新建项目最好的一条操作命令上传文件：
git init 
git add README.md 
git add .
git commit -m "first commit" 
git remote add origin git@github.com:wjp2018/vuecss.git
git push -u origin master

最后查看文件：git init




文件上传之后，以后要添加文件命令
git add .
git commit -m "first commit" 
git push -u origin master
最后查看文件：git init




错误提示：error: failed to push some refs to 
git push origin master
git push -u origin master
